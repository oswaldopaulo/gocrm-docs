GET - clientes

https://SEUSUBDOMINIO.gocrm.com.br/api/clients

    Campos requeridos

  • nome: String
  • contato_email: String
  • contato_nome: String
  • Token: String

    Todos os campos

  • nome: String
  • token: String
  • fone: String
  • celular: String
  • rua: String
  • numero: Number
  • bairro: String
  • cep_end: String
  • cidade: String
  • uf: String
  • cpf_cnpj: String
  • contato_email: String
  • contato_nome: Array
  • contato_email.*: String
  • contato_nome.*: String
  • contato_ramal: Array
  • contato_ramal.*: String
  • contato_telefone: Array
  • contato_telefone.*: String
  • endereco_descricao: Array
  • endereco_descricao.*: String
  • endereco_cep: Array
  • endereco_cep.*: String

Retornos HTTP:

200

GET - clientes

https://SEUSUBDOMINIO.gocrm.com.br/api/client/{id}

    Query params

  • type: String
  • query: String

Os valores do campo type podem ser: email, id, cpf_cnpj, Permitindo fazer buscas especificas no banco de dados.

Caso type seja omitido a busca padrão é por email, com retorno variado de local ou bling para onde o usuário está cadastrado.

Retornos HTTP:

200

PUT - clientes

https://SEUSUBDOMINIO.gocrm.com.br/api/client/update

    Campos requeridos

  • token: String
  • descricao: String
  • assunto: String
  • anexos: File

    Todos os campos

  • nome: String
  • email: String
  • idcliente: String
  • cpf_cnpj: String
  • Contato_email: String
  • Contato_nome: String
  • Token: String

Usamos como parâmetro para identificar o usuário em ordem os campos: email, idcliente, cpf_cnpj, na ausência resultará em um erro.

Retornos HTTP:

200

GET - produtos

https://SEUSUBDOMINIO.gocrm.com.br/api/products

    Query Params

  • id: String

Utilizamos o ID do chamado como parametro de busca

Retornos HTTP:

200

Retornos

Retorna informações de cada agenda para o chamado, contendo dados como: idgoogle onde verifica se a agenda está conectada ao Google.

Também contém o campo idfilhos onde verifica se o chamado é um chamado filho, ou seja, se este chamado está relacionado a outro chamado principal.
